Signomix

Signomix is an IoT platform allowing to the accumulation and using data from devices online LoRaWAN and other, communicating using HTTP.
The platform can be used both for testing prototypes, and for implementing commercial solutions

Basic functionalities include platforms:

  • managing IoT devices
  • integration with LoRaWAN networks
  • integration interface for Internet IoT solutions (Ethernet/WiFi)
  • REST API making the full functionality of the platform available
  • web application available to mobile devices and browsers desktopowych
  • defining desktops with the wide set of configurable indicator lights
  • definable enabling the data processing their analysis and the alteration and sending the message and orders to IoT devices
  • sending the message with e-mail channels, Slack, Pushover, Telegram, WebHook.

The online version, acting in the cloud is accessible at the https://signomix.com. Accounts free of charge are available to registered members of the association in the broadened version (details on the home page of the service).

A version Community Edition made available as the 100% open source on GitHub is accessible to people who would like to use the platform in the own plan and to start it in the personal infrastructure. It has an identical functionality as the version in the cloud, except of functions connected with paying for using as well as the section "Blog".

Our association is using the Signomix platform in the cloud in own plans and for the purposes of conducted workshops.

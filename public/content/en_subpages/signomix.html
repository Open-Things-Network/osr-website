Signomix

Signomix is an IoT platform enabling the collection and use of data from LoRaWAN devices and other devices using HTTP.
The platform can be used to test prototypes as well as to implement commercial solutions.

Basic platform functionalities include:

  • management of IoT devices
  • integration with LoRaWAN networks
  • integration interface for IoT solutions (Ethernet/WiFi)
  • REST API providing full platform functionality
  • web application available for mobile devices and desktop browsers
  • defining dashboards with a wide set of configurable controls
  • definable data processing enabling their analysis and modification as well as sending messages and commands to IoT devices
  • sending messages via e-mail, Slack, Pushover, Telegram, WebHook.

The online version, operating in the cloud, is available at signomix.com. Free accounts in the extended version are available for registered members of the association (details on the platform's home page).

There is a Community Edition version available as 100% open source on GitHub for people who would like to use the platform in their own project and run it in their own infrastructure. It has the same functionality as the cloud version, with the exception of features related to payments for use and the "Blog" section.

Our association uses Signomix cloud platform in its own projects and during the workshops.
